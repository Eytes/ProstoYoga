### Абонемент
Для каждого пользователя свой абонемент, на который начисляются занятия. Абонемент имеет срок жизни, так как занятия следует отходить до определенного момента времени
   Свойства   |   Описание   
 ------------- | ------------- 
 id | int
 id пользователя | int
 стоимость | int
 оставшиеся занятия | int
 до какого числа действителен | datetime
 
### Занятие
   Свойства   |   Описание   
 ------------- | ------------- 
 id | int
 
### Пользователь
   Свойства   |   Описание   
 ------------- | ------------- 
  id | int
  фото | blob
  Фамилия | str
  Имя | str
  Отчество | str
  Номер телефона | str
  Дата рождения | int

У каждого пользователя есть личный кабинет, в котором:
- фото пользователя
- информация пользователя
- информация о абонементе
- даты посещений
- возможность купить или продлить абонемент 

### Тренер
   Свойства   |   Описание   
 ------------- | ------------- 
 id | int

### Администратор
Так же имеет личный кабинет, как и обычный пользователь, но имеет другой функционал:
   Функционал   |   описание   
 ------------- | ------------- 
 управление занятиями  | - создать новый вид занятий<br>- изменить какое-то из существующих занятий<br>- удалить занятие  
 управление расписанием  | - создать дату занятий<br>- изменить дату занятий<br>- удалить дату занятий
 управление занятием | - создать занятие<br>- изменить занятие<br>- удалить занятие
