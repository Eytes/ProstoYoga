# ProstoYOGA

## Запуск сервера 
`uvicorn prostoyoga.main:app`

### Абонемент
Для каждого пользователя свой абонемент, на который начисляются занятия. Абонемент имеет срок жизни, так как занятия следует отходить до определенного момента времени
   Свойства | тип данных    
 --- | --- 
 id | int 
 id пользователя | int 
 осталось занятий | int 
 действителен до | datetime

### Пользователь
   Свойства   |   тип данных | Описание  
 --- | --- | ---
  id | int |
  фото | blob |
  Фамилия | str |
  Имя | str | |
  Отчество | str |
  Номер телефона | str |
  Дата рождения | int |

У каждого пользователя есть личный кабинет, в котором:
- фото пользователя
- информация пользователя
- информация о личном абонементе
- даты посещений
- возможность купить или продлить абонемент

### Администратор
   Функционал   |   описание   
 ------------- | ------------- 
 управление занятиями  | - создать занятие<br>- изменить какое-то из существующих занятий<br>- удалить занятие  
 управление расписанием  | - создать дату занятий<br>- изменить дату занятий<br>- удалить дату занятий
 управление тренерами | - бобавить нового тренера<br>- изменить информацию о тренере<br>- удалить тренера
