# ProstoYOGA

_Веб приложение для удобства ведения учета и контроля занятий_

## Сущности
[Абонемент](Абонемент)
- занятие
- скелет для пользователя
- пользователь
- суперпользователь

### Абонемент

   Свойства   |   Описание   
 ------------- | ------------- 
 id | int
 оставшиеся заняти | int
 

### Cкелет для пользователя
   Свойства   |   Описание   
 ------------- | ------------- 
  id | int
  Фамилия | str
  Имя | str
  Отчество | str
  Номер телефона | str
  Дата рождения | int


### Пользователь
Свойства наследуются от `Cкелет для пользователя`
   Свойства   |   Описание   
 ------------- | ------------- 
  id абонемента | int

У каждого пользователя есть личный кабинет, в котором:
- информация пользователя
- информация о абонементе
- даты посещений
- возможность купить или продлить абонемент 

### Суперпользователь


Так же имеет личный кабинет, как и обычный пользователь, но имеет другой функционал:
   Функционал   |   описание   
 ------------- | ------------- 
 управление занятиями  | - создать новый вид занятий<br>- изменить какое-то из существующих занятий<br>- удалить занятие  
 управление расписанием  | - создать дату занятия<br>- изменить дату занятия<br>- удалить дату занятия
